### 一、双向数据绑定

v-model指令可以实现数据的双向绑定，也就是说如果input标签添加v-model指令后，在页面上修改input标签内的文本也会直接改变v-model绑定的变量

```js
<body>
    <div class="app">
        <h1>数据绑定：{{msg}}</h1>
        <input type="text" v-model="msg">
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue"></script>
    <script>
        new Vue({
            el:".app",
            data:{
                msg:"123"
            }
        })
    </script>
</body>
```
