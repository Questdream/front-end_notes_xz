# 第09节：数据交互

### 知识点回顾

http协议：前端（浏览器）发送请求，服务器给予反应

请求方法：get（查询），post（添加），put（修改），delete（删除）

ajax：不刷新页面与后台交互数据

axios：封装好的ajax模块

koa：基于node的web框架

### 一、概述

基于vue的web项目显示的所有服务端数据都是通过ajax获取的，官方推荐使用axios作为ajax库

### 二、安装axios

在vue项目中使用`npm install axios --save`

在要引用axios的页面引用`import axios from "axios"`
